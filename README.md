## 🧠 About The Project

SkyBound is a **frontend travel website** built using pure HTML, CSS, and JavaScript.
It is designed to simulate a **real-world travel startup landing page**, focusing on UI structure, responsiveness, and scalability.

This project reflects how a modern travel platform presents destinations, offers, and services in a clean and engaging way.

---

## ✨ Features

* 🌍 Responsive design across all devices
* 🎨 Modern UI layout with structured sections
* 🧭 Smooth navigation experience
* 📄 Multi-page setup (About, Privacy, Terms)
* 💼 Services, Plans & Offers sections
* ⚡ Lightweight and fast performance

---

## 🛠️ Tech Stack

* HTML5
* CSS3
